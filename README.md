# SACE Mockup — HTML/CSS/JS

Esta pequeña demo reproduce la pantalla del mockup proporcionado: logo en escudo, botón "Bienvenido a SACE" y tres íconos inferiores en rojo.

Instrucciones rápidas (Windows PowerShell):

```powershell
# Abrir el archivo en el navegador predeterminado
Start-Process -FilePath "index.html"

# Alternativa: abrir con el Explorador de archivos
explorer.exe .\index.html
```

Archivos:
- `index.html` — estructura y SVG del logo
- `styles.css` — estilos para replicar la maqueta
- `script.js` — interacción mínima del botón
